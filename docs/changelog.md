# 更新日志

本文档记录塔罗牌小程序的版本更新历史、功能变更和问题修复。

## 版本命名规则

我们遵循[语义化版本](https://semver.org/lang/zh-CN/)命名规则：

- **主版本号**：当进行不兼容的API修改时
- **次版本号**：当添加功能并保持向下兼容时
- **修订号**：当进行向下兼容的问题修复时

## [1.0.0] - 2025-07-01（计划发布）

- 用户账号体系

  - 微信快捷登录
  - 用户信息管理
  - 会员等级系统

- 塔罗牌占卜系统
  - 单卡占卜
  - 三卡占卜

  - 马蹄型牌阵
  - 随机抽牌机制

- 塔罗牌解读系统
